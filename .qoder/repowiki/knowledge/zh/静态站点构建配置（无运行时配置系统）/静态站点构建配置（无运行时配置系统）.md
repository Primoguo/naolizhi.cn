该仓库是一个基于 React + Vite 的静态个人展示站点，**不存在运行时配置加载机制**。项目通过以下文件完成构建期与工具链配置：

- `vite.config.ts`：Vite 构建入口，仅定义 base、React 插件与 `@` 路径别名，未使用环境变量。
- `tailwind.config.js`：Tailwind CSS 主题扩展（字体、颜色、动画等），全部为硬编码值。
- `postcss.config.js`：PostCSS 插件列表（tailwindcss、autoprefixer）。
- `components.json`：shadcn/ui 组件库生成器的元数据，指定样式风格、别名映射等。
- `tsconfig.*.json`：TypeScript 编译选项。
- `eslint.config.js`：ESLint flat config。

代码中未发现任何对 `process.env`、`import.meta.env`、`.env*` 文件或运行时配置读取逻辑的使用，说明该站点不区分开发/生产环境，也不支持动态特性开关或外部化配置。所有行为均由源码和上述配置文件在构建时确定。
