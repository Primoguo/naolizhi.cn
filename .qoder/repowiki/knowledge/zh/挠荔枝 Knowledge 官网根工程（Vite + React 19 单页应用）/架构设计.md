- 构建与入口：`vite.config.ts` 通过 `@vitejs/plugin-react` 启用 React 支持并配置 `@/src` 路径别名；`src/main.tsx` 使用 `createRoot` 在 `<StrictMode>` 下挂载 `<App />`，`index.html` 为唯一 HTML 模板。
- 组件分层：`src/App.tsx` 作为页面编排器，自上而下组合 `sections/*`（FluidCanvas / EarParticles / Navbar / Hero / Features / Highlights / TTSDemo / CTA / Footer），每个 section 是独立的全屏或区块级展示组件；`components/ui/*` 存放由 shadcn/cli 生成的原子 UI 组件（button/card/tooltip）；`hooks/*` 封装可复用的动效 Hook（磁性吸附、滚动显示、聚光灯、移动端检测）；`lib/utils.ts` 提供共享工具函数。
- 视觉特效层：`FluidCanvas.tsx` 内嵌完整 GLSL shader（splat/advection/divergence/pressure/gradientSubtract/display）并通过 FBO 双缓冲实现 Navier-Stokes 流体模拟，配合 `IntersectionObserver` 在不可见时暂停渲染；`EarParticles.tsx` 负责耳形粒子装饰。
- 样式体系：Tailwind 3 + PostCSS + `tailwindcss-animate`，颜色/圆角/阴影全部映射到 CSS 变量（`hsl(var(--primary))` 等），`components.json` 声明 shadcn 的 new-york 风格与 `@/components/ui` 别名。
- 依赖方向：`sections` → `hooks` / `components/ui` / `lib`；`App` 仅做布局组合，不持有业务状态；无路由层，所有内容以同页区块呈现。
