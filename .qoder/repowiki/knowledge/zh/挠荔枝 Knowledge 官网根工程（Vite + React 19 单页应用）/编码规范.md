- 组件统一以默认导出函数形式定义，并在文件顶部用注释块标注模块职责（如 `// ===================== GLSL Shaders =====================`）。
- 样式完全走 Tailwind 原子类 + CSS 变量语义色（`bg-[#050505]`、`text-foreground`、`hsl(var(--primary))`），禁止在组件中写内联 style 以外的自定义 CSS。
- 副作用型逻辑封装为 `useXxx` 命名 Hook，通过 `useRef` 持有 DOM 引用、`useCallback` 缓存事件处理器，并在 useEffect 清理阶段移除 window 监听与 cancelAnimationFrame。
- WebGL 资源（FBO、Program、Texture）按 read/write 双缓冲模式组织，通过 `swap()` 交换读写端并在组件卸载时统一释放。
- 移动端降级策略集中在运行时判断（如 `window.innerWidth < 768` 跳过 WebGL），而非条件渲染整段代码，以保证桌面端体验不受影响。
