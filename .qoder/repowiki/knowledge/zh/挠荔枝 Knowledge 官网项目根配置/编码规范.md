- 路径导入统一使用 `@/` 别名（如 `@/components/ui`、`@/lib/utils`），由 vite resolve.alias 与 tsconfig paths 共同保障。
- 主题色、边框、阴影等视觉 token 全部通过 CSS 变量 (`var(--xxx)`) 暴露，Tailwind 中仅以 `hsl(var(--xxx))` 引用，避免硬编码颜色值。
- Dark mode 通过 `html.dark` class 控制，配合 `next-themes` 在运行时切换，而非媒体查询自动模式。
- React 组件遵循函数式 + TypeScript 接口定义 props 的模式，UI 组合优先复用 shadcn/ui 原子组件而非手写 DOM。
