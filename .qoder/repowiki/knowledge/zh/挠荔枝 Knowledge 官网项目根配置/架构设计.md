本模块为纯前端静态站点（无子模块），以 Vite 7 作为开发与构建工具，入口 `index.html` 挂载 `#root` 并通过 `<script type="module" src="/src/main.tsx">` 加载应用；`vite.config.ts` 通过 `@vitejs/plugin-react` 启用 React 支持并配置 `@/` → `./src` 路径别名，`base: './'` 适配相对路径部署。

- 样式体系：Tailwind CSS 3.4 + PostCSS + Autoprefixer，`tailwind.config.js` 使用 `darkMode: "class"` 配合 `next-themes` 实现暗色切换，主题色、圆角、阴影、动画 keyframes 均通过 CSS 变量 (`hsl(var(--xxx))`) 驱动，并引入 `tailwindcss-animate` 与自定义 `accordion-down`、`wave` 等动画。
- UI 组件：采用 shadcn/ui（`components.json` 声明 style=new-york、tsx=true、iconLibrary=lucide、aliases 指向 `@/components/ui` 等），结合 Radix UI 原子组件与 `class-variance-authority` / `clsx` / `tailwind-merge` 做变体与类名合并。
- 类型系统：TS 5.9 使用 Project References 拆分 `tsconfig.app.json`（浏览器端）与 `tsconfig.node.json`（Node/Vite 侧），顶层 `tsconfig.json` 仅维护 `@/*` 路径映射供 IDE 引用。
- 代码质量：ESLint Flat Config（`eslint.config.js`）集成 `@eslint/js`、`typescript-eslint`、`react-hooks`、`react-refresh`，忽略 `dist` 产物。
- SEO：`index.html` 内嵌 Open Graph、Twitter Card 与 Schema.org JSON-LD 结构化数据，目标域名 `https://naolizhi.cn`。
